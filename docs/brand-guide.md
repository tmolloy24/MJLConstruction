# MJL Renovations Brand Guide

## Positioning

MJL Renovations should feel like a premium design-build company rather than a general contractor. The brand should communicate organization, craftsmanship, transparency, and confidence without appearing flashy or overly corporate.

## Brand promise

**Thoughtfully planned. Carefully built. Made to last.**

## Primary messaging themes

1. Premium craftsmanship
2. Clear communication
3. Organized project management
4. Respect for the client's home
5. Transparent scope and expectations
6. One accountable construction partner

## Colour palette

| Role | Hex | Usage |
|---|---:|---|
| Primary charcoal | `#151515` | Header, footer, dark sections, headings |
| Secondary charcoal | `#242424` | Cards, overlays, secondary dark backgrounds |
| Warm white | `#F8F6F2` | Main page background |
| Pure white | `#FFFFFF` | Cards and high-contrast sections |
| Muted gold | `#B9985A` | Buttons, rules, icons, links, restrained accents |
| Body text | `#4B4B4B` | Paragraphs and supporting text |
| Light border | `#DED9CF` | Dividers, cards, forms |
| Soft grey | `#EEEAE3` | Alternate section backgrounds |

Gold should be used sparingly. The site should remain primarily charcoal, warm white, and photography-led.

## Typography

### Headings

Preferred: **Oswald**

Fallbacks: `Arial Narrow`, `Arial`, `sans-serif`

- Uppercase only for short labels, buttons, and eyebrow text
- Main page headings should use title case
- Avoid overly condensed body text

### Body

**Roboto**

Fallbacks: `Arial`, `sans-serif`

## Spacing system

Use an 8px base scale:

- 8px — micro spacing
- 16px — compact element spacing
- 24px — card spacing
- 32px — grouped content spacing
- 48px — component separation
- 72px — tablet section spacing
- 96px — standard desktop section spacing
- 120px — large hero or statement section spacing

## Container widths

- Main content width: 1240px
- Narrow copy width: 760px
- Wide gallery width: 1440px
- Page gutters: 32px desktop, 24px tablet, 20px mobile

## Buttons

### Primary

- Background: `#B9985A`
- Text: `#151515`
- Border: 1px solid `#B9985A`
- Padding: 16px 26px
- Border radius: 2px
- Font: Oswald, 15px, 600, uppercase, 0.7px letter spacing
- Hover: charcoal background, white text, charcoal border

### Secondary

- Transparent background
- Text and border adapt to section background
- Same spacing and typography as primary
- Hover fills with charcoal or white depending on context

## Imagery

- Use large, high-resolution finished-space photography
- Prioritize real MJL work whenever available
- Avoid generic construction-worker stock photos in primary sections
- Use project details, material texture, cabinetry, tile, millwork, lighting, and architectural features
- Hero images should provide open space for readable copy overlays
- Apply subtle dark overlays between 35% and 55% where text sits over images

## Voice

The copy should be direct, polished, and specific. Avoid inflated claims such as “best in the business,” “unmatched quality,” or “dreams into reality” unless supported by proof.

Preferred language:

- Carefully planned
- Built around your home and lifestyle
- Clear scope and communication
- Experienced trades
- Detailed project management
- Premium materials and finishes
- Respectful, organized worksites

## Core CTA language

Primary:

- Start Your Renovation
- Request a Consultation
- Get a Project Estimate

Secondary:

- View Our Work
- Explore the Process
- See Renovation Services

Avoid using “Learn More” when a more specific action can be used.
