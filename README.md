# MJL Construction — Renovations Website

This repository contains the design system, copy, Elementor build specifications, custom CSS, and implementation notes for the MJL Construction renovations website.

## Primary goal

Position MJL as a premium renovation and custom-home contractor serving Burlington, Oakville, Waterdown, Hamilton, Ancaster, Dundas, Milton, Flamborough, and Stoney Creek.

## Core services

- Kitchen renovations
- Bathroom renovations
- Basement renovations
- Whole-home renovations
- Full-gut renovations
- Custom-home building

## Technology

- WordPress
- Astra theme
- Elementor
- Elementor Pro
- Happy Elementor Addons
- Formidable Forms
- Yoast SEO

## Brand direction

- Premium, modern, image-led design
- Dark charcoal, warm white, and muted gold palette
- Strong typography and generous spacing
- Conversion-focused page structure
- Reusable Elementor sections

## Repository structure

- `docs/` — brand, sitemap, page copy, SEO, and implementation guidance
- `css/` — global styles and reusable component CSS
- `elementor/` — page-by-page Elementor build specifications
- `assets/` — logos, icon notes, and image requirements
